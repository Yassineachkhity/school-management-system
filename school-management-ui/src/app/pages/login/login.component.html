Secure access

School Management Portal

Sign in to manage students, teachers, courses, and enrollment from one workspace.

  • Centralized CRUD across all microservices
  • Role aware access for admin, teacher, and student
  • Keycloak based authentication and JWT flow
Configure Keycloak settings in src/environments/environment.ts.

Access flow

Gateway first

The UI calls the gateway on port 8888, and the gateway validates every request.

Roles in action

Admin manages everything, teachers grade and read, students read curriculum and enrollment.

Data sync

Admin can sync Keycloak users to the auth database for user tracking.
